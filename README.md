# Task 5: Train-Test Split & Evaluation Metrics

## Description
In this task, the Heart Disease dataset is split into training and testing sets. 
A Logistic Regression model is trained and evaluated using accuracy, precision, recall, and confusion matrix.

## Dataset
Heart Disease Dataset

## Model Used
Logistic Regression

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- Confusion Matrix

## Outcome
Understood how to evaluate machine learning models and interpret performance metrics.
